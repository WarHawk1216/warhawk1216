About the site

The site is created using Github Pages and uses a HTML5 UP template. I've experimented with Node JS and Three.js to create dynamic pages with 3D Models, but for now all models are on SketchFab until a new dynamic page is complete. The page is updated through Github Desktop and a composer JSON file.
